# ⚡ Bắt đầu nhanh

## 🎯 Chọn phiên bản phù hợp:

### 1️⃣ Phiên bản File JSON (Đơn giản - Khuyến nghị)
✅ Chạy ngay, không cần setup
✅ Dữ liệu lưu trên máy
✅ Phù hợp: Dùng 1 máy hoặc cùng WiFi

```cmd
npm install
npm start
```

### 2️⃣ Phiên bản MongoDB Atlas (Đám mây)
✅ Truy cập từ mọi nơi
✅ Tự động backup
✅ Phù hợp: Nhiều giáo viên, nhiều địa điểm

```cmd
npm install
npm run test:mongodb
npm run start:mongodb
```

**Lưu ý**: Cần setup MongoDB Atlas trước (5 phút)
📖 Xem: `SETUP_MONGODB_NHANH.md`

## 🚀 Sau khi chạy:

Truy cập: http://localhost:3000

## 📱 Cài đặt như App:

1. Nhấn nút "📱 Tải Ứng Dụng" ở góc dưới
2. Hoặc dùng tính năng "Cài đặt" của trình duyệt

## 🌐 Chia sẻ với đồng nghiệp:

### Cùng WiFi:
1. Nhấn "🔗 Chia sẻ" trong app
2. Quét QR Code

### Khác WiFi:
Deploy lên internet (xem `docs/DEPLOY_NHANH.md`)
